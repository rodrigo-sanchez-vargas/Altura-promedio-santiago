# Análisis de alturas

## Descripción: El código analiza un archivo CSV para extraer sus datos y, a partir de su lectura, respecto a alturas genéricas generadas mediante Copilot, realizar el cálculo de altura promedio tanto para hombres como para mujeres.

## Lenguaje: Python + librerías numpy y matplotlib.pyplot.

## Conocimientos utilizados: Manejo de archivos, ciclos y gráficos.
